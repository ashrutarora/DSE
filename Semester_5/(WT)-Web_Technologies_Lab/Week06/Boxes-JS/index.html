JavaScript Boxes

Alert Box - The alert() function displays a pop-up dialog box with a specified message and an "OK" button.

It's often used to show informational messages to the user.

Prompt Box - The prompt() function displays a dialog box that prompts the user for input.

It can be used to collect data from the user.

Confirm Box - The confirm() function displays a dialog box with a message and "OK" and "Cancel" buttons.

It's often used to get a yes/no response from the user.

Line Break - The document.write() method can be used to write content to the document.

In this example, it adds a line break after some text.
